The library was verified to work with:

`ESP8266`, `ESP32`, `MKR1000`, `Particle Photon`, `Particle Electron`, `nRF51`, `nRF52`, `Teensy 3.1/3.2`

**Note:** it requires at least **~64Kb** Flash and **~10Kb** RAM even for minimal functionality.
